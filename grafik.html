Grafik Fungsi

Yuk masukkan fungsi yang ingin kamu coba ke kolom "Masukkan Fungsi" dan lihat grafiknya muncul seketika! tapi jangan lupa, perhatikan cara penulisan fungsi dan keterangan agar hasilnya tepat ya! belajar jadi lebih seru dan interaktif bersama Granyda!.

Penulisan Fungsi

Fungsi Linear : y = ax + b

Fungsi Kuadrat : y = ax^2 + bx + c

Fungsi Kubik : y = ax^3 + bx^2 + cx + d

Fungsi Eksponen : y = a^x

Fungsi Logaritma : y = log(a)

Keterangan

a, b, c, d = Nilai a, b, c, d

x = Variabel x

+ = Tambah

^ = Pangkat
